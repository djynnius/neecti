Usage in Tailwind CSS

The new primary green color can be used in your React components:

<!-- Background color --> <div className="bg-primary-green">Green background</div> <!-- Text color --> <span className="text-primary-green">Green text</span> <!-- Border color --> <div className="border-primary-green">Green border</div> <!-- CSS Classes --> <button className="btn-success">Success Button</button> <button className="btn-outline-success">Outline Success</button>

Suggested Use Cases

  • Success states: Form submissions, successful actions
  • Positive actions: "Save", "Confirm", "Accept" buttons
  • Status indicators: Online status, verified badges
  • Progress indicators: Completed steps, achievements
  • Call-to-action: Secondary CTAs that need attention
